# Free Backlinks Generator

WordPress theme for community guest posting and backlink exchange.

## Install

Copy this repository into `wp-content/themes/free-backlinks-generator`, then activate the theme in **Appearance → Themes**.

## Documentation

- `seo.md`, `remote-management.md`, `theme-build-instructions.md`, `CHANGELOG.md`

The theme creates required pages on activation and registers the `fbg_post` post type with the public archive at `/community/`.
