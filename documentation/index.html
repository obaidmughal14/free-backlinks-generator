Requirements

  • WordPress 6.0 or newer (tested through 6.7 in theme header).
  • PHP 8.0 or newer.
  • HTTPS recommended for member login, cookies, and checkout links.

Installation

  1. Unzip the theme package. Upload the folder free-backlinks-generator to wp-content/themes/ (do not nest a second copy inside itself).
  2. In Appearance → Themes, activate Free Backlinks Generator.
  3. On activation, the theme may create required pages (home, login, register, dashboard, etc.). Visit Settings → Permalinks and click Save once to flush rewrite rules if URLs 404.
  4. Go to Appearance → Customize to set branding, colors, typography, and business URLs (Pro checkout, footer text).

For ThemeForest: include this documentation folder in your main ZIP or provide it as a separate download per marketplace rules.

Customizer — “Theme options (Community)” panel

Open Appearance → Customize. All buyer-facing options live under the panel Theme options (Community) plus separate sections where noted.

Colors — light mode

Maps to CSS variables on :root (primary, accent, backgrounds, text). Use these to rebrand without editing CSS files.

Colors — dark mode

Used when the visitor enables dark mode (or system preference). Primary stays dark for sidebars and footer bands.

Typography & fonts

Choose a preset pairing (Google Fonts) or select Custom and paste a full https://fonts.googleapis.com/css2?... URL from Google Fonts. For custom URLs, set the display and body font family names to match the loaded families.

Layout

  • Max content width — controls --fbg-container-max (960–1320px).
  • Corner radius — compact, balanced, or roomy presets for cards and buttons.

Site behavior

  • Show dark / light mode toggle — disable if you want a single scheme only.
  • Open external links in a new tab — applies to post content, widgets, and menu items pointing off-site.

Footer text

Tagline after the copyright, optional designer credit line (lead text, name, URL).

Other sections (same panel or global)

  • Logo size — responsive max width/height for header and footer logos.
  • Homepage stats — four numbers under the marketing hero.
  • Pro upgrade — checkout URLs — Stripe / WooCommerce links for the member dashboard.
  • Footer — column headings — titles above footer menu columns.
  • Social & discoverability — social URLs and optional homepage meta description.

Menus & key templates

Assign menus under Appearance → Menus to these locations:

LocationUse
Header menuPrimary navigation on inner pages.
Footer — column 1–3Three footer columns (titles set in Customizer).

Community posts use the fbg_post post type; archives and singles are styled by the theme.

Child theme

Included sample: child-theme-fbg/ — copy this folder to wp-content/themes/, rename if you like, then activate the child theme. Keep the parent folder name as free-backlinks-generator so the Template: header matches.

Uncomment the enqueue block in child-theme-fbg/functions.php if you add CSS to the child style.css.
