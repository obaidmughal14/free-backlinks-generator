# Changelog

## 1.0.0 — 2026-04-02

- Initial release of the Free Backlinks Generator WordPress theme.
- Custom post type `fbg_post` with archive at `/community/`, niches and content types.
- Member role `fbg_member`, registration, login, forgot password, dashboard, and submit flows.
- AJAX archive filters, load more, notifications table, CSV and GDPR JSON export.
- Design system CSS, template parts, and bundled demo WXR for one sample post.
